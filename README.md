# EC-sensor
모듈화된 EC센서 코드입니다(장현석)

## EC sensor 작동방식
1. 최초 작동시 5칸의 빈 배열이 만들어지고 센서에서 측정한 값이 할당됩니다.
2. 배열이 전부 할당되면 그 5개의 값들의 평균을 내고 다음 측정값이 평균의 1.5배인지 측정합니다.
3. 만약 1.5배라면 경고 메시지와 부저가 경고를 합니다.
4. 아니라면 그 값이 배열에 할당이 됩니다.(FIFO 자료구조 활용)
5. 그리고 만약 값이 0이라면 배열에 할당되지 않고 통과합니다.
